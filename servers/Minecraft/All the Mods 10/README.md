## How to Join

1. Message `@zuedev` on Discord with your Minecraft username to be whitelisted on the server.
2. Install the modpack using your preferred Minecraft launcher. We recommend using the [Prism Launcher](https://prismlauncher.org/) for the best experience.
3. Ensure that you have the correct version of the modpack (v5.4) and that all mods are properly installed.
4. Start the game and select the "Multiplayer" option from the main menu.
5. Click "Add Server" and enter the following details:
   - **Server Name:** Unnamed Group ATM10
   - **Server Address:** `atm10.play.unnamed.group`
6. Click "Done" and then select the server from your multiplayer list to join.
